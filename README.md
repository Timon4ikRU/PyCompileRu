# PyCompileRu - препроцессор для Python, написанный на Python.

## Описание
Компилятор русских ключевых слов Python в обычный Python код. Позволяет писать программы на русском языке, что более удобно для школ и/или людей не знающих/плохо знающих английский язык.

## Как запустить код? 
```bash
python run_ru.py test.ru.py
```
или, *если не сработала первая команда*,
```bash
python3 run_ru.py test.ru.py
```

## Пример программы
```python
# test.ru.py
from pyru import *

если __name__ == "__main__":
    имя = строка(ввести("Как тебя зовут? \n "))
    # или просто "имя = ввести("Как тебя зовут? \n")
    печатать("Привет,", имя)
```

## Зачем расширение .ru.py? 
Вообще, .ru.py - просто конвенция, но есть варианты:
1. **Ясно указывает** что это russian-python код
2. **Не конфликтует** с обычными .py файлами
3. **Легко фильтровать** *какие* файлы компилировать

Либо же можно сделать так: 
```text
src/
  program.ru.py    # исходник
dist/
  program.py       # скомпилированный
```

## Поддерживаемые ключевые слова
- `если` → `if`
- `для` → `for` 
- `функция` → `def`
- и многие другие...

## Требования
- Python 3.6+

## Структура проекта
```text
pycompileru/  
├── pyru.py # Русские функции (печатать, ввести, и т.д.)  
├── pycompileru.py # Компилятор русских ключевых слов  
├── run_ru.py # Запускатель программ  
├── test.ru.py # Пример программы  
└── README.md # Этот файл  
```

## Установка
Просто скачайте последний релиз данного скрипта. По возможности, будут выпускаться пре-релизы, однако НЕ РЕКОМЕНДУЕТСЯ переносить свой код на такую версию пока она не станет обычным релизом.
